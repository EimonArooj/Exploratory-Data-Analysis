# 📊 Week 2 – Exploratory Data Analysis (EDA) | Applicants Dataset

## 📌 Project Overview

This project was completed as part of **Week 2** of the **Data Science Internship at NextgenLearners**.

The objective of this task was to perform **Exploratory Data Analysis (EDA)** on the Applicants dataset to uncover meaningful patterns and trends using Python. The analysis focuses on applicant domains, application trends, application status, university participation, and submission behavior through various visualizations.

---

## 🎯 Objectives

- Explore the applicants dataset using Pandas.
- Analyze application distribution across different domains.
- Study application trends over time.
- Examine the overall application status.
- Compare application status across different domains.
- Identify universities contributing the highest number of applicants.
- Create an additional visualization based on an independent analytical question.
- Summarize key findings through actionable insights.

## 📊 Analysis Performed

The notebook includes the following analyses:

- Number of Applications by Domain
- Monthly Application Trend
- Overall Application Status Breakdown
- Application Status by Domain
- Top Universities by Number of Applicants
- Applications by Day of the Week (Bonus Analysis)

Each visualization includes:
- A clear business question
- Appropriate chart labels and titles
- A brief interpretation of the results

---

## 📈 Key Findings

- Data Science received the highest number of applications, making it the most preferred domain among applicants.
- Application activity reached its peak during May and remained high in June before declining in July.
- Nearly half of all applications are still under review, while around one-fifth have already been selected.
- FAST NUCES, NUST, and Punjab University contributed the largest number of applicants.
- Friday recorded the highest number of application submissions, indicating increased applicant activity toward the end of the academic week.
